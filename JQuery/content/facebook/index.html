Set Up Notes:

Set the fbAdminID variable to the admin actor_id for the site. This allows the code to know which posts are public posts and which are admin posts. To get the actor_id, do the following:

  1. Set includePublicFeed and fbShowControls to false in the page script to ensure no public feeds will be included.
  2. Ensure fbInit() and fbFetch() are called and reload the page.
  3. Do one of the following using the browser Console:
    • Use the facebook query url in a new browser tab. Look for actor_id in the results and use the value to set fbAdminID.
    • Within the Console or Net tabs, view the response sent from the facebook request. Look for actor_id in the results and use the value to set fbAdminID.
  4. Restore/Remove includePublicFeed and fbShowControls based on the site preferences.


* Plugin must be accompanied by a DreamStudio link, or be hosted by an approved site.
